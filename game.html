🍴 How to Play

  1. Read the customer's order card 📋
  2. Click the correct ingredients from the grid 🛒
  3. Press Start Cooking! to validate your ingredients 🍳
  4. Click each action button in the right order (Chop → Mix → Cook…) 👨‍🍳
  5. Hit Serve Dish! to earn points — faster = bigger bonus! 🍽
Keyboard shortcuts: C = Chop  ·  M = Mix  ·  K = Cook  ·  B = Blend  ·  P = Plate  ·  Space = Start Cooking  ·  Enter = Serve  ·  Esc = Clear
