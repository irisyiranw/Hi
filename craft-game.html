🖌️ How to Play

  1. Read the project card to see what to make 📋
  2. Click the correct supplies from the grid 🛒
  3. Press Start Crafting! to begin making it 🎨
  4. Click each step button in the right order (Sketch → Color → Decorate…) ✨
  5. Hit Show Off! to earn points — faster = bigger bonus! 🌟
Keyboard shortcuts: S = Sketch  ·  C = Color  ·  X = Cut  ·  F = Fold  ·  G = Glue  ·  D = Decorate  ·  Space = Start Crafting  ·  Enter = Show Off  ·  Esc = Clear
